Задание 1.

Исходные данные: var a = 1, b = 1, c, d;
1) c = ++a; alert(c); // Результат: 2 .
Объяснение: К переменной 'а' со значением '1' применяется преинкрементация, после чего результат присваивается переменной 'с'. Т.е сначала значение 'а' увеличивается на 1, после чего присваивается 'c'.
2) d = b++; alert(d); // 1
Объяснение: К переменной 'b' со значением '1' применяется постинкрементация, после чего результат присваивается переменной 'd'. Т.е сначала значение 'b' присваивается переменной 'd', после чего 'b' увеличивается на 1.
3) c = (2+ ++a); alert(c); // 5
Объяснение: Сначала 'а' увеличивается на 1 и становится равно 3, после чего прибавляется еще 2. 3+2=5.
4) d = (2+ b++); alert(d); // 4
Объяснение: Сначала к 'b' со значением '2' прибавляется 2, после чего результат (2+2=4) присваивается 'd'. 'b' после этого увеличивается на 1.
5) alert(a); // 3
alert(b); // 3
Объяснение: Изначально 'a' и 'b' равнялись 1. В ходе выполнения операций к обоим переменным еще два раза была применена инкрементация. Следовательно, 1+1+1=3 для обеих переменных.
